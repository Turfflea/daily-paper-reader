## 摘要
文本属性图（TAG）广泛应用于许多现实世界领域，在TAG上进行学习需要联合建模文本语义和图结构。建模TAG的一种标准方法是结合语言模型（LM）和图神经网络（GNN），但联合训练计算成本高昂且难以扩展。数据集蒸馏是降低训练成本的一种有前景的方法，但现有方法并不适合TAG，因为它们通常是为单一模态设计的，或者在蒸馏过程中仍需在完整数据集上重复训练昂贵的LM-GNN模型。为了解决这个问题，我们提出了TaLK，一种有效的TAG数据集蒸馏方法，它将LM与图感知神经正切核相结合。这种设计实现了高效的数据集蒸馏，避免了在完整数据集上进行重复的联合训练，同时反映了文本和结构信息以实现有效的TAG学习。在多个TAG基准测试上的实验表明，TaLK始终优于现有基线，并仅使用1%的合成数据即可达到完整数据集性能的97%。

## Abstract
Text-attributed graphs (TAGs) are widely used in many real-world domains, and learning on TAGs requires jointly modeling text semantics and graph structure. A standard approach for modeling TAGs is to combine a language model (LM) and a graph neural network (GNN), but joint training is computationally expensive and difficult to scale. Dataset distillation is a promising way to reduce training costs, but existing methods are not well suited to TAGs because they are typically designed for a single modality or still require repeatedly training expensive LM-GNN models on the full dataset during distillation. To address this, we propose TaLK, an effective dataset distillation method for TAGs that couples an LM with a graph-aware neural tangent kernel.This design enables efficient dataset distillation, avoiding repeated joint training on the full dataset while reflecting both textual and structural information for effective TAG learning.Experiments on multiple TAG benchmarks show that TaLK consistently outperforms existing baselines and achieves up to 97% of full-dataset performance with only 1% synthetic data.

### 2. 方法论
- **整体框架**：TaLK 采用双层优化。内层在合成数据集上联合训练 LM 与 GNN；外层通过图感知神经正切核（graph‑aware NTK）与核岭回归（KRR）来优化合成数据，**避免外层直接训练 GNN**，从而大幅降低计算与内存开销。
- **关键技术细节**：
  - **合成数据初始化**：从原始训练集中按类平衡采样 M 个节点，将其文本属性的 token 嵌入作为初始合成特征 X′；合成邻接矩阵固定为单位矩阵 I（结构‑free 蒸馏）。
  - **内层优化**：在 (X′, I) 上联合微调 LM（LoRA）和 GNN（GCN），使 LM 参数受 GNN 梯度驱动，保持与最终训练一致的联合训练行为。
  - **外层优化**：使用内层训练后的 LM 对全量 TAG 和合成数据分别提取节点表示 HT、HS，基于结构感知 NTK（SNTK）计算核矩阵 KSS、KTS，通过 KRR 得到全量节点上的标签预测，并以有标签节点的预测误差作为外层损失。
  - **内存高效实现**：对 SNTK 进行两项改进：(i) 利用合成图 A′=I 将 Kronecker 积的邻域聚合化简为直接在原图邻接矩阵上的乘法；(ii) 用特征范数近似自核矩阵对角元素，避免存储完整自核矩阵。
  - **Batch‑wise 梯度注入**：外层损失基于全局合成表示计算，梯度按 batch 切片后注入各 mini‑batch 的 LM 反向传播，在保持全局优化目标的前提下实现可扩展的 mini‑batch 训练。

### 6. 主要结论与发现
- TaLK 在所有数据集和压缩比下均显著优于 coreset 选择和解耦蒸馏基线。

- 在超参搜索任务上，TaLK 比全数据训练加速 **12.6×∼17.7×**，且总搜索时间与解耦蒸馏基本持平。

### 7. 优点（亮点）
- **首次实现 TAG 端到端联合蒸馏**：在保持 LM–GNN 联合训练范式的同时，蒸馏过程自身无需反复训练大型 LM–GNN。
- **内存高效设计**：通过精简的 SNTK 实现与 batch‑wise 梯度注入，使方法能在有限 GPU 内存上应用于较大规模 TAG。
- **理论与实现紧密结合**：利用核岭回归提供闭式解，避免外层 GNN 训练；梯度注入保留了全局优化性质。
- **全面的实验验证**：覆盖多个领域、多种 backbone，并提供跨架构、超参搜索等实用场景评估。

- **可解释性**：蒸馏结果停留在 token 嵌入层级，无法直接生成可读文本，应用场景受限。
