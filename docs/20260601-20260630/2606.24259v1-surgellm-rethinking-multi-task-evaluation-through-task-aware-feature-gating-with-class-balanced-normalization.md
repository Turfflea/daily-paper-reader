## 摘要
部署在异构NLP任务中的微调编码器面临三个复合问题：不匹配的归纳偏置、类别不平衡对特征统计的破坏，以及缺乏根据外部词汇知识调节注意力的机制。我们引入了 SURGELLM，一个统一的transformer框架，通过专门的轻量级模块来解决每个问题：一个“外科特征门”（在精选的词汇指示符和[CLS]上学习每个维度的sigmoid函数；当特征无信息时可证明退化为恒等映射），任务条件化前缀令牌（量化的特征值和任务标识符预置到每个输入），以及实例加权归一化（IWN；从门统计中移除类别先验偏置）。我们证明了一个超额风险界，将门的益处与外科特征对齐联系起来。在四个任务上，包括SST-2、多跳检索、LLM提示归因和作者检测，覆盖17,830个示例和11个模型变体，使用三个随机种子，IWN变体实现了宏观F1值0.940（比最强的非IWN基线高0.036；在作者检测上高0.130）。随机词汇对照（平均F1下降0.028）证实增益来自词汇而非参数。代码、词汇表和一个99.5%恢复率的自动提取配方已发布。

## Abstract
Fine-tuned encoders deployed across heterogeneous NLP tasks face three compounding problems: mismatched inductive biases, class-imbalance corruption of feature statistics, and no mechanism to condition attention on external lexical knowledge. We introduce \textbf{\surgellm}, a unified transformer framework that addresses each with a dedicated lightweight module: a \emph{surgical feature gate} (learned per-dimension sigmoid over curated lexical indicators and \texttt{[CLS]}; provably degenerates to identity when features are uninformative), \emph{task-conditioned prefix tokens} (quantized feature values and task identity prepended to every input), and \emph{Instance-Weighted Normalization} (IWN; removes class-prior bias from gate statistics). We prove an excess-risk bound linking gate benefit to \emph{surgical feature alignment}. Across four tasks, SST-2, multi-hop retrieval, LLM-prompt attribution, and authorship detection, covering 17,830 examples and eleven model variants over three seeds, the IWN variant achieves macro-F1 \textbf{0.940} ($+0.036$ over the strongest non-IWN baseline; $+0.130$ on authorship detection). A random-vocabulary control ($-0.028$ avg.\ F1) confirms gains are lexical, not parametric. Code, vocabularies, and a $99.5\%$-recovery auto-extraction recipe are released.

### **1. 论文核心问题与研究背景**

论文旨在解决在**结构异构的NLP任务**上进行**多任务学习**时，共享编码器面临的三个复合性问题：

-   **归纳偏置不匹配**：不同任务（如情感分析、多跳检索、作者检测）依赖的表层语言信号差异巨大，简单的端到端微调无法最优利用所有信号。
-   **类别不平衡导致特征统计污染**：传统的全局特征标准化（Global Normalization）在训练数据类别分布严重倾斜时，其统计量会偏向多数类，污染注入模型的特征信号，误导特征融合过程。
-   **缺乏外部词汇知识引导**：预训练编码器可能不擅于捕捉或充分利用一些关键的、全局的词汇级别统计信息（如特定词汇密度、标点频率等），这在高判别性的任务（如作者风格检测）中尤为关键。

核心假设是，通过向编码器显式地、有条件地注入经过精心设计的词汇与表层统计特征，并纠正类别不平衡带来的统计偏置，能够显著提升多任务学习的整体表现。

### **2. 方法与关键技术**

论文提出了 **首字母缩略词框架** 框架，一个与任何HuggingFace编码器兼容的即插即用模块化架构。其核心组件如下：

-   **外科特征门：**
    -   **特征提取**：基于一个包含10组词汇指示符的词表，从完整、未被截断的原始文本中提取一个16维的统计特征向量 **s(x)**。
    -   **特征融合**：将该特征向量 `s` 经过非线性投影得到 `s'`，然后与编码器的 `[CLS]` 表示 `h` 在维度上进行拼接。通过一个可学习的、逐维度的 `sigmoid` 门控机制 `g`，对 `[CLS]` 和 `s'` 进行加权融合。
    -   **公式表达**：`g = σ(W_g · [h̃; s']) ，最终的融合表示为 ĥ = LayerNorm(g ⊙ h̃ + (1-g) ⊙ s')`。该设计允许模型在不同维度上自主决定对语义信息（来自 `[CLS]`）和词汇统计信息（来自 `s'`）的依赖程度。

-   **任务条件化前缀令牌：**
    -   在原始输入文本前，拼接一段可解释的、结构化的文本前缀（例如：`[TASK: Sentiment | F1:v1 | … | F16:v16]`）。
    -   该前缀将任务标识和量化后的特征值以自然语言Token的形式注入，使得模型的所有自注意力层都能在上下文词元层面感知到任务类型和词汇特征，与在最终 `[CLS]` 层起作用的门控机制形成互补。

-   **实例加权归一化：**
    -   **问题**：在类别不平衡严重的数据集上，计算全局均值和标准差的传统标准化方法会使特征 `s` 的分布偏向多数类，导致门控网络学习到次优的融合策略。
    -   **解决方案**：IWN**在训练时**，首先计算每个类别的特征均值和标准差，然后取这些类别统计量的简单平均作为全局的标准化参数（`ŝ(x) = (s(x) - s̄_bal) / σ_bal`）。
    -   **核心优势**：这是一种**无参数、测试时不依赖标签**的方法。它通过均衡化类别先验，确保特征 `s` 的分布是对称的，从而使门控网络能公平地从所有类别中学习。
