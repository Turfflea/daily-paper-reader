## 摘要
自监督学习（SSL）已成为从未标记数据中学习有意义表示的强大范式。然而，评估这些表示的标准方法——线性探测——计算成本高、对超参数敏感，并且对表示空间的几何结构提供的洞察有限。在本工作中，受神经网络泛化与内在维度（ID）之间联系的启发，我们提出了IdEst，一种通过最小生成树维度估计器（dim_MST）估计SSL表示ID的方法。在多种数据集、架构和SSL预训练目标上，我们表明IdEst与下游线性探测性能强相关。此外，我们证明IdEst能够实现高效的超参数选择，与有监督替代方案相比显著降低了计算成本。我们的结果凸显了内在维度作为评估SSL表示的原则性几何代理，补充了标准的有监督探测协议。

## Abstract
Self-supervised learning (SSL) has emerged as a powerful paradigm for learning meaningful representations from unlabeled data. However, the standard protocol for evaluating these representations, linear probing, is computationally expensive, sensitive to hyperparameters, and provides limited insight into the geometric structure of the representation space. In this work, motivated by connections between neural network generalization and intrinsic dimension (ID) we propose IdEst, a method for estimating the ID of SSL representations via the Minimum Spanning Tree dimension estimator ($\mathrm{dim}_\mathrm{MST}$). Across diverse datasets, architectures, and SSL pretraining objectives, we show that IdEst strongly correlates with downstream linear probe performances. Furthermore, we demonstrate that IdEst enables efficient hyperparameter selection, significantly reducing the computational cost compared to supervised alternatives. Our results highlight intrinsic dimensionality as a principled geometric proxy for assessing SSL representations, complementing standard supervised probing protocols.

### 2. 方法论
- **核心思想**：基于定理 3.1（泛化误差界 $\mathcal{L}\simeq\mathcal{O}(K_L N_D^{-1/d})$），**内在维度 $d$ 越低，表示越高效**。因此，直接估计表示空间的 $d$ 即可作为下游性能的预测指标。
- **选用估计器 $\mathrm{dim}_{\mathrm{MST}}$**：
  - 传统估计器（TwoNN、MLE）依赖局部各向同性假设，在 SSL 的非渐近、高维、样本依赖等条件下表现不稳定（见图 2，TwoNN 甚至发散）。
  - $\mathrm{dim}_{\mathrm{MST}}$ 基于最小生成树（MST）长度的渐近增长率与 Rényi 熵的关系，对噪声、采样密度变化、高环境维数具有强鲁棒性，尤其在 $n\approx d$ 的稀疏采样下有效。
- **IdEst 算法流程**（见 Algorithm 1）：
  - 输入：冻结的 SSL 表示集合 $Z=\{z_1,\dots,z_N\}$。
  - 生成递增大小 $n_i$ 的子样本，计算对应 MST 的总边长 $L(\mathrm{MST}(Z_{n_i}))$。
  - 对 $\log L$ 与 $\log n_i$ 进行线性回归，得到斜率 $m$。
  - 内在维度估计值 $d = 1/(1-m)$。

### 6. 主要结论与发现
- **强负相关**：IdEst 与线性探测准确率呈显著负相关（Spearman’s $\rho\approx-0.8$，Kendall’s $\tau\approx-0.6$），低内在维度是高质表示的几何标志。
- **跨场景推广**：该相关性不仅在同数据集内成立，在跨数据集（例如在 ImageNet 上计算 IdEst，在 iNat/CIFAR 上评估）以及不同评估协议（kNN、v2）下同样保持。
- **训练动态追踪**：在离线与在线探测设置中，IdEst 随预训练 epoch 增加而单调下降，与下游准确率提升同步，可作为训练进程的无监督指示器（findings 2）。
- **高效超参数选择**：IdEst 在完全无标签条件下表现出与监督 Oracle 可比的模型选择能力，且优于 $\alpha$-ReQ、RankMe；在非联合嵌入范式（如 I‑JEPA）中鲁棒性更强。
- **计算高效**：相比 10‑epoch 线性探测，IdEst 计算时间降低一个数量级以上，尤其在大模型上优势明显。

### 7. 优点
- **无需标签**：彻底摆脱了对监督信号的需求，可在大批量预训练模型中快速筛选。
- **几何可解释性**：从流形内在维度角度解释表示质量，补充了有效秩（RankMe）和均匀性指标的不足。
- **估计器鲁棒性**：$\mathrm{dim}_{\mathrm{MST}}$ 基于 MST 的渐近性质，对高维、稀疏采样、噪声等更稳定，优于流行的 TwoNN 与 MLE。
- **方法通用**：不仅适用于联合嵌入 SSL，也适用于联合预测和视觉‑语言模型，展现了跨范式的适应力。
- **资源友好**：仅需单次特征提取与相对轻量的图计算，适合大规模模型的原型评估。

### 8. 不足与局限
- **效能非完美预测**：内在维度与性能的相关性虽强但并非一一对应，当模型准确率紧密聚集时排序能力下降；视觉‑语言模型（CLIP 等）因模态隔阂（锥效应）出现系统性偏差。
- **早期训练不适用**：在 ViT 预训练的前 10 个 epoch 内，表示几何结构尚未稳定，IdEst 不能提供有效信号。
- **理论绑定而非等式**：定理 3.1 提供的仅为收敛速率上界，两个内在维度相近的模型实际泛化能力仍可能不同。
- **应用范围受限**：当前实验限于图像分类的全局表示评估，尚未扩展到密集预测任务（分割、生成）等场景，理论框架也需要相应扩展。
- **仍需子采样**：IdEst 仍需在大型数据集上做子采样（n=50k）与 MST 计算，虽快于线性探测，但并非零成本。
